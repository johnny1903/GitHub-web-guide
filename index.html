🧑‍💻 Hướng dẫn Public Trang Web HTML lên GitHub Pages (Dễ Hiểu)

🔧 Bước 1: Tạo tài khoản GitHub

Truy cập https://github.com và nhấn Sign Up để tạo tài khoản miễn phí.

🔧 Bước 2: Tạo repository mới

  1. Đăng nhập GitHub
  2. Nhấn dấu “+” → New repository
  3. Đặt tên repo, ví dụ: my-first-web
  4. Chọn Public và tick vào Add a README file
  5. Nhấn Create repository

🔧 Bước 3: Thêm file HTML

  1. Trong repo vừa tạo, chọn Add file → Create new file
  2. Đặt tên file là index.html
  3. Dán nội dung HTML của bạn
  4. Nhấn Commit new file

🔧 Bước 4: Kích hoạt GitHub Pages

  1. Vào tab Settings của repo
  2. Chọn mục Pages (trong sidebar trái)
  3. Trong phần Source, chọn:
    • Branch: main
    • Folder: / (root)
  4. Nhấn Save
✅ Sau vài phút, trang của bạn sẽ hoạt động tại:
https://[tên-tài-khoản].github.io/[tên-repo]/

🎉 Xong rồi!

Bạn có thể sửa file index.html bất cứ lúc nào để cập nhật nội dung website.
